Dashcord

This page contains the case study of Dashcord which includes the Project Overview, Tools Used and Live Links to the official product.

Project Overview

This Web Application is a Student performance monitor developed to streamline communication between teacher’s and Students. It is a platform where teacher’s can store all the related data of the students which comes under them. This Web Application includes few Dashboards, those Dashboards are Teacher, Teacher Incharge, Student and Head of Department (HOD). The Teacher Incharge will assign some students to the teacher’s and after assigning the students Teacher’s will add those students to their respective Dashboards and monitor their performance.
